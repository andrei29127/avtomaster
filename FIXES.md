Я не смог внести изменения прямо в GitHub-репозиторий из этого чата, но подготовил готовые файлы для замены.

Файлы в архиве:
- package.json
- next.config.ts
- README.md

Что менять у себя:
1. Замени эти файлы в корне проекта.
2. Удали bun.lock.
3. Выполни:
   npm install
   npm run typecheck
   npm run build

Если после этого останется ошибка, следующим шагом нужно чинить src/app/page.tsx или конкретный API route.
